A slight variant of the traditional OO implementation, which removes the explicit, 'old fashioned (i.e. not idiomatic to Javascript)' Observer/Subject classes from which Models and Controllers inherit - in favour of a Javascript native pub/sub mechanism, like document.dispatchEvent() from subject functions and document.addEventListener() on observing functions
